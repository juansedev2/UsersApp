db_users.sql contiene el código SQL exportado desde el SGBD MariaDB versión 10.4.27, así que se recomienda usar este mismo SGBD en lo posible y tan solo sería necesario importarlo para obtener tanto la base de datos, su estructura como la data existente. Exiten dos versiones, la primera contiene la versión inicial con algunos usarios ya registrados para hacer pruebas, la segunda estará más completa de usuarios, manteniendo la misma estructura.
