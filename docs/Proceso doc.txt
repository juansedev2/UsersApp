Acá estará un poco de documentación acerca de este proyecto:

LoginApp es una aplicación que puede realizar las siguientes funciones:

1. Permite realizar registro e inicio de sesión de usuarios.
2. Dado un usuario administrador, se pueden realizar operaciones CRUD con otros usuarios.
3. Dado un usuario que no sea administrador, este podrá ver opciones de navegación para ver vistas estáticas y ver su información y el único dato que puede cambiar sería su contraseña.

Es más que todo como un ejercicio para una aplicación del patrón de arquitectura MVC.

Puede considerarse su uso como base a una aplicación más grande (considerando su modificación para su adaptación).

El proceso de documentación del proyecto está dado en un documento llamado "Documentación desarrollo.pdf" que es un simple proceso de documentación del desarrollo de la App. En proyectos de software por lo general se suele incluir esta parte de manera más formal (plantillas, estándares, metodologías, etc.), así que tomarlo el existente como un ejemplo simple, ya que no amerita la complejidad de realizarlo, más se deja en observación la importancia de contar con documentación siempre, tanto interna como externa.
